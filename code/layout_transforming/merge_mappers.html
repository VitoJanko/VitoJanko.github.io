make_unmerge_default_mapper(labels, values, values_merged, wide=False, backwards=False, ignore_merged=False, enforce_unique=True, merge_model_upwards=False, status=None)

This function creates a default mapper for unmerging cells in a table. It takes into account various parameters such as whether to ignore merged cells, enforce uniqueness, and whether to merge model upwards.

Parameters:

  • labels (DataFrame) – The labels of the table.

  • values (DataFrame) – The values of the table.

  • values_merged (DataFrame) – The merged values of the table.

  • wide (bool) – Optional parameter to specify if the table is wide.

  • backwards (bool) – Optional parameter to specify if the iteration should be backwards.

  • ignore_merged (bool) – Optional parameter to specify if merged cells should be ignored.

  • enforce_unique (bool) – Optional parameter to specify if uniqueness should be enforced.

  • merge_model_upwards (bool) – Optional parameter to specify if model should be merged upwards.

  • status (DataFrame or None) – Optional parameter to specify the status of the table.

Returns:

The cell mapper.

Return type:

CellMapper
