helpers

check_for_mandatory_categories(category_to_columns, mandatory_categories: list)

Checks if all mandatory categories are present in the given category_to_columns dictionary. If a mandatory category is missing, an exception is raised.

Parameters:

  • category_to_columns (dict) – Dictionary with categories as keys.

  • mandatory_categories (list) – List of mandatory categories.

Raises:

Exception – If a mandatory category is missing in category_to_columns.

find_table_above_current_one(table_metadata, tables: List[TableData])

Finds a table that is located above the current one and has the same width. If no such table is found, returns None.

Parameters:

  • table_metadata (dict) – Metadata of the current table.

  • tables (List[TableData]) – List of all tables.

Returns:

The table located above the current one and has the same width, or None if no such table is found.

Return type:

TableData or None

use_above_table_header(table_data, all_tables) → pd.DataFrame | None

If a table is found above the current one and has the same number of columns, the header of the above table is used for the current table. If no such table is found, returns (False, None).

Parameters:

  • table_data (TableData) – Data of the current table.

  • all_tables (list) – List of all tables.

Returns:

A tuple where the first element is a boolean indicating if the operation was successful, and the second element is the updated table or None.

Return type:

tuple

is_table_above_and_same_width(table_1, table_2)

Checks if table_2 is located above table_1 and has the same width. Returns a tuple where the first element is a boolean indicating if table_2 is above and has the same width as table_1, and the second element is the top coordinate of table_2 or None.

Parameters:

  • table_1 (dict) – The first table.

  • table_2 (dict) – The second table.

Returns:

A tuple where the first element is a boolean indicating if table_2 is above and has the same width as table_1, and the second element is the top coordinate of table_2 or None.

Return type:

tuple
