Returns a map of categories to their corresponding values extracted from the table.

Parameters:

  • categories (list) – List of categories to extract values for.

  • table_metadata_block (dict) – Metadata of the table to extract data from.

  • category_to_textract_blocks_map (dict) – Map of categories to their corresponding textract blocks.

  • tolerance (float) – Tolerance for the extraction process.
