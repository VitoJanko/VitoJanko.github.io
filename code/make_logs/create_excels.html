create_excel(destination: LogDestination, match_results: MatchAllResults, eq_results: EquipmentResults, pl_tables: list[TableParser], td_tables: list[TechnicalParser])

Parameters:

  • destination (LogDestination) – The destination where the Excel file will be saved.

  • match_results (MatchAllResults) – The results of the match.

  • eq_results (EquipmentResults) – The results of the equipment.

  • pl_tables (list[TableParser]) – The list of PL tables.

  • td_tables (list[TechnicalParser]) – The list of TD tables.

None
