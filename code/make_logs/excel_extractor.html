extract_labels(file_index, input_folder='excels', output_name='cell_labels', output_layout_name='layout', generate_extra=False, generate_generic=None, input_name='annotated')

This function extracts labels from an Excel file.

Parameters:

  • file_index (int) – The index of the file.

  • input_folder (str) – The folder where the input file is located. Default is “excels”.

  • output_name (str) – The name of the output file. Default is “cell_labels”.

  • output_layout_name (str) – The name of the output layout file. Default is “layout”.

  • generate_extra (bool) – Whether to generate extra labels. Default is False.

  • generate_generic (None or list) – Whether to generate generic labels. Default is None.

  • input_name (str) – The name of the input file. Default is “annotated”.

Returns:

The extracted labels.
