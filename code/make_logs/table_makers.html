draw_table(values, labels, margin_row, margin_column, sheet, background=None, title='', splitters=None, border=True, fill_background=None, wrap_around=False)

Draw a table in a given sheet with specified values, labels, margins, and other parameters.

Parameters:

  • values (pandas.DataFrame) – The values to be included in the table.

  • labels (pandas.DataFrame) – The labels for the values in the table.

  • margin_row (int) – The margin for the rows.

  • margin_column (int) – The margin for the columns.

  • sheet (openpyxl.worksheet.worksheet.Worksheet) – The sheet where the table will be drawn.

  • background (str or None) – The background color for the cells in the table.

  • title (str) – The title of the table.

  • splitters (list[int] or None) – The indices of the columns that should be split.

  • border (bool) – Whether to add a border to the cells in the table.

  • fill_background (str or None) – The background color for the cells in the table.

  • wrap_around (bool) – Whether to wrap the text in the cells around.

Returns:

None
