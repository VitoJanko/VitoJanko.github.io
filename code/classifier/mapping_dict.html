mapping_dict

get_mapping_dict(price_list_id, categories_of_interest=[])

Generate a dictionary mapping values to categories based on a given price list.

Parameters:

  • price_list_id (int) – The ID of the price list to use for the mapping.

  • categories_of_interest (list[str], optional) – Optional list of categories to include in the mapping. If not provided, all categories are included.

Returns:

The mapping dictionary.

Return type:

dict

get_full_mapping_dict(price_list_id_list, categories_of_interest=[])

Generate a full mapping dictionary for a list of price lists and categories of interest.

Parameters:

  • price_list_id_list (list[int]) – The list of price list IDs to use for the mapping.

  • categories_of_interest (list[str], optional) – Optional list of categories to include in the mapping. If not provided, all categories are included.

Returns:

The full mapping dictionary.

dict
