get_best_combination_from_duplicates(word: str, word_combinations: list[str], word_combinations_original_indices: list[list[int]])

From all duplicates of word (combination) keep only one. Select the one which has source words closest to each other.

Parameters:

  • word (str) – The word to find the best combination for.

  • word_combinations (list[str]) – The list of word combinations.

  • word_combinations_original_indices (list[list[int]]) – The original indices of the word combinations.

Returns:

The index of the best combination and the list of duplicate indices.

Return type:

(int, list[int])
