make_line_reference(text_to_line: dict, line_block: dict)

This function appends the line block to the list of line blocks corresponding to the text in the text_to_line dictionary. If the text is not already a key in the dictionary, it is added.

Parameters:

  • text_to_line (dict) – A dictionary where the keys are texts and the values are lists of line blocks.

  • line_block (dict) – A dictionary representing a line block.

Returns:

None

adobe_to_textract(adobe_output: tuple, original_pages: list[int])

This function converts Adobe output to Textract format. It processes the Adobe output and creates a Textract dictionary with the corresponding metadata, blocks, and page relations.

Parameters:

  • adobe_output (tuple) – A tuple containing Adobe JSON and CSVs.

  • original_pages (list[int]) – A list of original page numbers.

Returns:

A dictionary in Textract format.

Return type:

dict
