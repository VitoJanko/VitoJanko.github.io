merge_versions(equipments: list, combined_versions_to_id_map: dict, versions_to_id_map: dict)

Merges and delete versions in a list of equipment.

Conditions: - Merge/delete will only occur if the number of versions in the equipment is greater than the number of versions in the main price list. - Versions selected for merge/delete are only those with fewer instances of equipment than the base version on the price list.

Heuristics for merging and deleting versions after the above condition is met: - A merge will occur if the version name is contained in another version”””s name. Example: “””(MY22)””” will merge into “””Initiale paris (MY22)”””. - A version will be deleted under two conditions:

  1. The version name is not contained in any other version.

2. The version has less than half the number of equipment instances of the base version.

Parameters:

  • equipments (list) – A list of Equipment instances.

  • combined_versions_to_id_map (dict) – A dictionary of version name to version ID for all versions.

  • versions_to_id_map (dict) – A dictionary of version name to version ID for main versions.

Returns:

A modified list of Equipment instances.

Return type:

list
