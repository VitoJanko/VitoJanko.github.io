process_layout

make_columns(pages, segment_maker: TextractDivider, pdf_parser: PdfParser, debug: bool)

This function generates columns from the given pages using the provided segment maker and PDF parser.

Parameters:

  • pages (list) – The pages to generate columns from.

  • segment_maker (TextractDivider) – The segment maker to use for generating columns.

  • pdf_parser (PdfParser) – The PDF parser to use for generating columns.

  • debug (bool) – A boolean indicating whether to print debug information.

Returns:

A tuple containing the generated columns, a dictionary mapping pages to columns, and a list of equipment.

Return type:

(list[list[EquipmentLine]], dict[int, int], list)
