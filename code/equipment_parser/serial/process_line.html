process_line

add_titles(lines: list[EquipmentLine], serial_attributes: SerialAttributes, debug: bool)

Add titles to the lines based on certain conditions. If a line meets at least two conditions, it is considered a title.

Parameters:

  • lines (list[EquipmentLine]) – List of equipment lines.

  • serial_attributes (SerialAttributes) – Serial attributes of the lines.

  • debug (bool) – If True, print debug information.

Returns:

The list of lines with titles added.

Return type:

list[EquipmentLine]

classify_lines(lines: list[EquipmentLine], classifier: EQModel, serial_attributes: SerialAttributes, debug: bool)

Classify the lines based on the predictions from the classifier. The lines can be classified as color, misc, package, or equipment.

Parameters:

  • lines (list[EquipmentLine]) – List of equipment lines.

  • classifier (EQModel) – The classifier model.

  • serial_attributes (SerialAttributes) – Serial attributes of the lines.

  • debug (bool) – If True, print debug information.

analyze_lines(lines: list[EquipmentLine], columns: dict[int, int], debug) → SerialAttributes

Analyze the lines and return the serial attributes. The attributes include page margins, sizes, color usage, dash usage, and special dash.

Parameters:

  • lines (list[EquipmentLine]) – List of equipment lines.

  • columns (dict[int, int]) – Dictionary mapping page numbers to column numbers.

  • debug (bool) – If True, print debug information.

Returns:

The serial attributes of the lines.

Return type:

SerialAttributes

combine_lines(lines: list[EquipmentLine], use_dashes: bool, serial_attributes: SerialAttributes, debug: bool) → list[EquipmentLine]

Combine the lines based on certain conditions. If a line starts with a lower case letter and the previous line starts with an upper case letter, or if the previous line starts with a dash and the current line does not, the lines are combined.

Parameters:

  • lines (list[EquipmentLine]) – List of equipment lines.

  • use_dashes (bool) – If True, use dashes for line joining.

  • serial_attributes (SerialAttributes) – Serial attributes of the lines.

  • debug (bool) – If True, print debug information.

Returns:

The list of combined lines.

Return type:

list[EquipmentLine]
