extract_price_simplified(x, default_return_value=None)

Extracts the price from a string. If the string does not contain a price, it returns the default return value.

Parameters:

  • x (str) – The string to extract the price from.

  • default_return_value (Any) – The value to return if the price cannot be extracted.

Returns:

The extracted price or the default return value.

float or Any
