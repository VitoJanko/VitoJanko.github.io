eq_packages

create_package(eq: Equipment, lines: list[str] = None)

Create a package of equipment based on the title of the equipment. The function splits the title into different elements and creates a package for each element.

Parameters:

  • eq (Equipment) – The equipment to be packaged.

  • lines (list[str] or None) – Optional list of lines to be used for splitting the title. If not provided, the title is split based on certain characters.

Returns:

A list of packaged elements.

Return type:

list

parse_according_to_type(eq: Equipment, eq_model: EQModel, lines: list[str] = None) → list

Parse the equipment according to its type predicted by the equipment model. If the type is “package”, it creates a package. If the type is “color”, it returns a color object. Otherwise, it returns the equipment itself.

Parameters:

  • eq (Equipment) – The equipment to be parsed.

  • eq_model (EQModel) – The model used to predict the type of the equipment.

  • lines (list[str] or None) – Optional list of lines to be used for splitting the title. If not provided, the title is split based on certain characters.

Returns:

A list of parsed elements.

Return type:

list
