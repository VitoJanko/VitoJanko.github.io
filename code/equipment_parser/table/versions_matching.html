versions_matching

match_versions_to_ids_V3(col_id_to_value: dict[int, str], versions_to_id_map: dict[str, int], additional_versions_to_id_map: dict[str, int], brand_id: int)

Match versions to ids and return a list of matched versions, average score and a boolean indicating if textract should be used.

Parameters:

  • col_id_to_value (dict[int, str]) – A dictionary mapping column ids to values.

  • versions_to_id_map (dict[str, int]) – A dictionary mapping versions to ids.

  • additional_versions_to_id_map (dict[str, int]) – An additional dictionary mapping versions to ids.

  • brand_id (int) – The id of the brand.

Returns:

A tuple containing a list of matched versions, the average score and a boolean indicating if textract should be used.

Return type:

tuple[list[VersionMatch], float or None, bool]
