detect_possible_splitted_table(values: pd.DataFrame, versions: list[str], versions_additional: list[str], settings: Settings)

Detects if a table is possibly split into multiple parts.

Parameters:

  • values (pd.DataFrame) – The DataFrame containing the table values.

  • versions (list[str]) – List of versions.

  • versions_additional (list[str]) – Additional list of versions.

  • settings (Settings) – Settings for the detection.

Returns:

A tuple containing a boolean indicating if the table is split, the split indices, and the DataFrame of versions.

Return type:

tuple

Splits a table into multiple parts based on the split indices.

Parameters:

  • values_blocks (pd.DataFrame) – The DataFrame containing the table blocks.

  • split_indices (list[tuple]) – Indices where the table should be split.

  • df_versions (pd.DataFrame) – DataFrame of versions.

Returns:

A tuple containing the new values and new values blocks.

tuple
