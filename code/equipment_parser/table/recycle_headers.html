recycle_headers

generate_header(values: pd.DataFrame, versions: list) → pd.DataFrame

Generates a header for a DataFrame by adding versions to the end of the header row.

Parameters:

  • values (pd.DataFrame) – The input DataFrame.

  • versions (list) – The list of versions to be added to the header row.

Returns:

A new DataFrame containing the header row with versions. Returns None if the number of columns in values is less than or equal to the length of versions.

Return type:

pd.DataFrame

concat_header(previous_header, values_original, values_blocks)

Concatenates the header row to the given DataFrames.

Parameters:

  • previous_header (pd.DataFrame) – The header row to be added.

  • values_original (pd.DataFrame) – The original DataFrame.

  • values_blocks (pd.DataFrame) – The DataFrame containing blocks of values.

Returns:

A tuple containing the updated DataFrames with the header row added.

Return type:

tuple

try_previous_header(previous_header, values, values_original, values_blocks)

Attempts to merge the previous header with the given DataFrames.

Parameters:

  • previous_header (pd.DataFrame) – The header row to be added.

  • values (pd.DataFrame) – The input DataFrame.

  • values_original (pd.DataFrame) – The original DataFrame.

  • values_blocks (pd.DataFrame) – The DataFrame containing blocks of values.

Returns:

A tuple containing a boolean indicating the success of the merge, and the updated DataFrames with the header row added if successful.

Return type:

tuple

recycle_header(values, values_original, values_blocks, versions_to_id_map, previous_headers, settings)

Tries to recycles the previous header rows by trying to merge it with the given DataFrames.

Parameters:

  • values (pd.DataFrame) – The input DataFrame.

  • values_original (pd.DataFrame) – The original DataFrame.

  • values_blocks (pd.DataFrame) – The DataFrame containing blocks of values.

  • versions_to_id_map (dict) – A mapping of versions to their IDs.

  • previous_headers (list) – A list of previous header rows.

  • settings (object) – The settings for the recycling process.

Returns:

A tuple containing a boolean indicating the success of the merge, and the updated DataFrames with the header row added if successful.

Return type:

tuple
