get_serial_eq_pages(eq_pages: list[int], page_to_eq_count: dict, page_to_eq_count_textract: dict)

This function returns a list of page numbers that have less than 3 equations, based on the counts from Adobe and Textract.

Parameters:

  • eq_pages (list[int]) – List of page numbers to check.

  • page_to_eq_count (dict) – Dictionary mapping page numbers to equation counts from Adobe.

  • page_to_eq_count_textract (dict) – Dictionary mapping page numbers to equation counts from Textract.

Returns:

List of page numbers with less than 3 equations.

Return type:

list[int]
