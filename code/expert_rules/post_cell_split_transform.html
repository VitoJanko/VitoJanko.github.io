post_cell_split_transform

post_cell_split_transform(values: pd.DataFrame, labels: pd.DataFrame, settings: Settings, summary: Summary)

Depending on the brand and country specified in the summary, restructures the versions in the values and labels dataframes.

Parameters:

  • values (pd.DataFrame) – The dataframe containing the values.

  • labels (pd.DataFrame) – The dataframe containing the labels.

  • settings (Settings) – The settings for the transformation.

  • summary (Summary) – The summary containing the brand and country.

restructure_versions(values: pd.DataFrame, labels: pd.DataFrame, column='version')

Restructures the versions in the values and labels dataframes based on the specified column.

Parameters:

  • values (pd.DataFrame) – The dataframe containing the values.

  • labels (pd.DataFrame) – The dataframe containing the labels.

  • column (str) – The column based on which the versions are restructured. Default is “version”.
