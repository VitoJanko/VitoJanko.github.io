transform_values_labels(values: pd.DataFrame, values_merged: pd.DataFrame, labels: pd.DataFrame, status: pd.DataFrame, settings: Settings, summary: Summary, labeler: CellLabeler)

Apply transformations to the values and labels based on the settings and summary. The transformations include label transformation, neo patentati exception, repair prices, versions to one cell, repair versions, join rows, fill dacia drivetrain, parse kw for electric cars, join rows on merged cells.

Parameters:

  • values (pd.DataFrame) – The dataframe of values.

  • values_merged (pd.DataFrame) – The dataframe of merged values.

  • labels (pd.DataFrame) – The dataframe of labels.

  • status (pd.DataFrame) – The dataframe of status.

  • settings (Settings) – The settings for transformation.

  • summary (Summary) – The summary of the data.

  • labeler (CellLabeler) – The labeler for cells.

Returns:

The list of changes made.

Return type:

list
