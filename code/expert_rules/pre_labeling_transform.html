pre_labeling_transform

transform_values(values: pd.DataFrame, status: pd.DataFrame, settings: Settings, summary: Summary)

Transforms the values in a DataFrame based on expert rules. If a value is changed, the corresponding status is updated.

Parameters:

  • values (pd.DataFrame) – DataFrame of values to be transformed.

  • status (pd.DataFrame) – DataFrame of status corresponding to the values.

  • settings (Settings) – Settings for the transformation.

  • summary (Summary) – Summary of the data.

Returns:

None

value_transformation(value: str, summary: Summary, settings: Settings) → Tuple[str, bool]

Transforms a single value based on expert rules. The transformation includes removing certain prefixes, replacing certain words, and removing brand and model names.

Parameters:

  • value (str) – The value to be transformed.

  • summary (Summary) – Summary of the data.

  • settings (Settings) – Settings for the transformation.

Returns:

The transformed value and a boolean indicating whether the value was changed.

Return type:

Tuple[str, bool]
