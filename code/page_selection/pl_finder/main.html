prepare_tables_full(page: trp.Page, page_number, labeler, settings, language, return_all, textract_results, summary, is_technical_details=False, is_equipment=False)

Prepare tables with different content types and append them to a list if they meet certain conditions.

Parameters:

  • page (trp.Page) – The page to process.

  • page_number (int) – The number of the page.

  • labeler (object) – The labeler to use.

  • settings (object) – The settings to use.

  • language (str) – The language of the page.

  • return_all (bool) – Whether to return all tables or not.

  • textract_results (dict) – The results from Amazon Textract.

  • summary (object) – The summary of the document.

  • is_technical_details (bool) – Whether the page contains technical details or not.

  • is_equipment (bool) – Whether the page contains equipment details or not.

Returns:

The list of tables.
