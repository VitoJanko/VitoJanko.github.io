label_table(df_values: pd.DataFrame, transformation_type: str = 'default', model_type: str = 'area', storage: PreloadedAssets = None, ignore_code=True, ignore_model=False, return_proba=False) → pd.DataFrame

Apply a model to a dataframe to generate predictions.

Parameters:

  • df_values (pd.DataFrame) – The dataframe to be labeled.

  • transformation_type (str) – The type of transformation to apply to the data. Default is “default”.

  • model_type (str) – The type of model to use for prediction. Default is “area”.

  • storage (PreloadedAssets) – The storage object containing preloaded assets.

  • ignore_code (bool) – Whether to ignore code cells. Default is True.

  • ignore_model (bool) – Whether to ignore model cells. Default is False.

  • return_proba (bool) – Whether to return probabilities. Default is False.

Returns:

The labeled dataframe.

Return type:

pd.DataFrame
