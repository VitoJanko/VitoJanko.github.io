Check if there are two price lists in the pages.

Parameters:

  • pl_pages (list[int]) – List of page numbers where price list is found.

  • eq_pages (list[int]) – List of page numbers where equipment list is found.

bool

sort_pages(pl_pages, td_pages, eq_pages)

Sort and combine the pages, giving priority to pl and eq over td. Avoids duplicate pages.

Parameters:

  • pl_pages (list[int]) – List of page numbers where price list is found.

  • td_pages (list[int]) – List of page numbers where table of contents is found.

  • eq_pages (list[int]) – List of page numbers where equipment list is found.

Returns:

Sorted list of pages with their types.

list[tuple[str, int]]

detect_and_split_multi_price_lists(pl_pages, td_pages, eq_pages)

Detects if there are multiple price lists and splits them if found.

Parameters:

  • pl_pages (list[int]) – List of page numbers where price list is found.

  • td_pages (list[int]) – List of page numbers where table of contents is found.

  • eq_pages (list[int]) – List of page numbers where equipment list is found.

Returns:

Tuple containing a boolean indicating if multiple price lists were found and a list of tuples containing grouped page numbers for each price list and equipment list.

tuple[bool, list[tuple[list[int], list[int]]]]
