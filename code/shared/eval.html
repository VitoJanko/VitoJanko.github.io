partial_match(value_1: str, value_2: str, min_ratio=0.8, return_score=False)

Checks if the first string is in the second string or if the overall match ratio is greater than 0.9.

Parameters:

  • value_1 (str) – The first string to compare.

  • value_2 (str) – The second string to compare.

  • min_ratio (float) – The minimum ratio for a match. Default is 0.8.

  • return_score (bool) – If True, the function will return the match ratio. Default is False.

Returns:

True if there is a match, False otherwise. If return_score is True, it also returns the match ratio.

Return type:

bool or tuple

match_combination(true_comb, pred_comb, eval_mode, categories=[], missing_categories=None, pred_index=0, true_index=0) → bool

Matches two combinations based on certain criteria.

Parameters:

  • true_comb (dict) – The true combination.

  • pred_comb (dict) – The predicted combination.

  • eval_mode (str) – The evaluation mode.

  • categories (list) – The categories to consider for the match. Default is an empty list.

  • missing_categories (list or None) – The categories that are missing. Default is None.

  • pred_index (int) – The index of the predicted combination. Default is 0.

  • true_index (int) – The index of the true combination. Default is 0.

Returns:

True if the combinations match, False otherwise.

Return type:

bool
