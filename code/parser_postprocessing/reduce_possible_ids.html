get_data_from_table(table, regex_expression, type_cast)

Extracts data from a table using a regular expression and casts it to a specific type.

Parameters:

  • table (Table) – The table from which to extract data.

  • regex_expression (str) – The regular expression used to find matches in the table.

  • type_cast (function) – The function used to cast the matched data to a specific type.

Returns:

The list of data extracted from the table.

Return type:

list

Extracts technical data from a document using regular expressions.

Parameters:

  • textract_results (Document) – The results of the textract operation on the document.

  • pages (list[int]) – The pages of the document to extract data from.

Returns:

Two lists of data, one for kilowatt data and one for cubic centimeter data.
