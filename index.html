Library for extracting features in activity recognition

A collection of methods that streamline the activity recognition pipeline. The main focus is on motion and physiological feature extraction, but it also contains helper methods for signal pre-processing and prediction post-processing. With these methods we won first place in the Sussex-Huawei Locomotion Challenge in 2018 and 2019.

Feature extraction, signal processing, python package

A library for energy-efficient activity recognition

Sensor settings (e.g., sensor type, frequency, duty-cycling ...) can be changed to increase the energy efficiency of a given activity recognition system. Methods in this library use multi-objective optimization in order to (semi-)automatically find the most efficient settings. The novel algorithms in this library provided the basis for my PhD thesis.

Markov chains, multi-objective optimisation, pyplot, python package

AI for Progressive Chess

A chess variant where white makes one move, then black makes two in a row, white then makes three, etc. The developed system uses a modified A* algorithm to enhance Min-Max and was the first to beat a human in this variant.

AI for Briskula

A system for playing the briskula (briscola) card game. This system is comprised of three components: computer vision to detect cards, a robot arm that moves them and an AI that can play the game at a super-human level.

AI for drafting MTG

When playing Magic: The Gathering (card game) in "draft" mode, eight players first assemble a deck by iteratively picking cards. In this project, I used the data of thousands of past online matches to create an AI that can approximate human picks.
