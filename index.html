Library for extracting features in activity recognition

A collection of methods that streamline the activity recognition pipeline. The main focus is on motion and physiological feature extraction, but it also contains helper methods for signal pre-processing and prediction post-processing. Using these methods we achieved the first place in the Sussex-Huawei Locomotion Challenge in 2018 and 2019.

A library for energy-efficient activity recognition

Sensor settings (e.g., sensor type, frequency, duty-cycling...) can be changed to increase the energy efficiency of a given activity recognition system. Methods in this library use multi-objective optimization in order to (semi-)automatically find efficient settings. The novel algorithms in this library provided the basis for my PhD thesis.

Markov chains, multi-objective optimisation, pyplot, python package
